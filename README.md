# BrainBurst Quiz App

BrainBurst is a fun, fast-paced general knowledge quiz app built entirely with HTML, CSS, and JavaScript. It was created as a personal project to practice web development skills and build something genuinely useful and enjoyable.

## What is BrainBurst?

BrainBurst challenges players to answer 10 randomly selected questions from a pool of 25, covering topics like science, geography, history, technology, and mathematics. Each question has a 15-second countdown timer, keeping the game exciting and fast-paced.

## Features

- 25 questions with 10 randomly selected each game so every round feels fresh
- 15 second countdown timer per question
- Green highlight for correct answers and red for wrong answers
- Live score tracking throughout the quiz
- Progress bar showing how far through the quiz you are
- Top 5 leaderboard saved locally so you can track your best scores
- Clean and modern dark blue UI design
- Fully responsive layout

## How to Play

1. Open the app in your browser
2. Enter your name in the input box
3. Click the Let's Go button to start
4. Answer each question before the timer runs out
5. See green for correct and red for wrong answers
6. Complete all 10 questions to see your final score
7. Check the leaderboard to see your ranking
8. Click Play Again to try and beat your score

## Technologies Used

- HTML5 for structure and layout
- CSS3 for styling, animations and responsive design
- Vanilla JavaScript for all game logic including timer, scoring, random question selection and leaderboard

## Project Structure

- index.HTML - main structure and all screen layouts
- style.css - all styling and visual design
- script.js - all game logic and interactivity

## Challenges Faced

- Implementing random question selection without repeating questions
- Building an accurate countdown timer that resets between questions
- Creating smooth transitions between home, quiz and result screens
- Saving and displaying leaderboard data using localStorage
- Making answer buttons show correct and incorrect feedback visually

## What I Learned

Building BrainBurst taught me how to structure a multi-screen web app without any frameworks, manage state with JavaScript variables, manipulate the DOM dynamically, work with timers and callbacks, and store data locally in the browser.
